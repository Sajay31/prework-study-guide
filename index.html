HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML defines any extra attributes on an element
  • Attributes must be defined in the in the beginning, never closing

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branche we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push: adds the changes made to git hub

JavaScript

  • JavaScript is a program used by developers for making applications and web browsers
  • A variable is a container that is named and has the ability to store data within the code
  • console is a test environment used to check out code
